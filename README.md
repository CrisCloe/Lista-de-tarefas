# Lista-de-tarefas

Programa de lista de tarefas para obtenção da terceira nota da disciplina Engenharia de Software

Programa para listar tarefas e visualizar seu status, utilizando os pais de design Strategy (para definir uma família de algoritmos e colocar cada um deles em uma classe separada - adicionar tarefas; Listar tarefas; remover tarefa; Alterar status - e tornar seus objetos intercambiáveis.) e Singleton guarda todas as tarefas e garante que existe apenas uma lista de tarefas no programa (garante que uma classe tenha apenas uma instância). Os padrões foram utilizados logo no início do código (foram destacados respectivamente, primeiro o singleton e logo após a estratégia)
